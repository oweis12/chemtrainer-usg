# Virtual Labs — voorbereidingsmap

Deze map is bedoeld voor latere mini-labs binnen ChemTrainer USG. In deze ronde wordt alleen voorbereid; er wordt geen nieuw analyse-lab gebouwd.

## Mogelijke mini-labs later

- Massaspectrometrie
- GC / GC-MS
- IR
- NMR

## Architectuurprincipe

- Scene component: tekenen en beperkte interactie.
- Engine/logica: scheikundige regels, simulatie-uitkomst en foutfeedback.
- Lesson wrapper: uitleg, begrippen en koppeling aan de bestaande leeromgeving.
- Quiz layer: korte checkvragen en beoordeling.

## Voorwaarden voordat een nieuw lab wordt gebouwd

Begin pas met een nieuw Virtual Lab wanneer deze onderdelen stabiel zijn:

- practice scheduler;
- vraag-specifieke hints;
- TitratieLab-integratie;
- bestaande routes en build.

- Geen MassSpectrometryLab bouwen.
- Geen GC/IR/NMR-componenten bouwen.
- Geen nieuwe grote route of redesign toevoegen.
